# 📊 Sales Analysis Dashboard (Power BI)

## 🔍 Overview

This project is an interactive **Sales Analysis Dashboard** built using Power BI to analyze sales performance across different regions, products, and time periods. It helps in identifying key trends, top-performing areas, and business opportunities.

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing products and regions
* Track monthly and yearly sales trends
* Enable data-driven decision making

## 🛠️ Tools Used

* Power BI
* Microsoft Excel (Data Source)

## 📈 Key Features

* 📌 Total Sales, Profit, and Quantity KPIs
* 📊 Sales breakdown by Region and Category
* 📅 Time-based analysis (Monthly/Yearly trends)
* 🎯 Top-performing products visualization
* 🎛️ Interactive filters and slicers

## 📊 Key Insights

* Certain regions contribute the highest share of sales
* Seasonal trends observed in monthly sales
* Top products generate a major portion of revenue
* Some categories show lower performance and need attention
